Supporter Referral Program — Concept

A simple, transparent referral system that rewards advocacy.

---

🎁 Referral Rewards

1 Referral
- Referral badge  
- Dashboard flair  

3 Referrals
- Spotlight eligibility  
- Early access token  

5 Referrals
- Advocate badge  
- Personalized thank‑you  

10 Referrals
- Legacy Advocate badge  
- Recognition in annual report  

---

🔗 Referral Mechanics
- Each supporter receives a unique referral code  
- New supporters can optionally enter a code  
- Referral count updates automatically  

---

🧩 Governance Alignment
- No financial incentives  
- Recognition‑based only  
- Transparent tracking  
`
