# Funding the Ecosystem

This project is 100% community-funded. Your support directly enables development, audits, and contributor rewards.

## How to Fund
- **GitHub Sponsors:** Our primary channel. Offers perks and recognition.
- **Stripe:** For one-time donations via our [website](https://yourdomain.com/donate).

## Where Funds Go
Funds are distributed transparently via our automated payroll system. See the `/contracts/payroll` directory for details.
